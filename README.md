# Arduino Touch LCD Controller

Touch-based LED mode controller using:
- Arduino Uno
- Pushbutton
- 16x2 I2C LCD
- Red LED
- Green LED
- Blue LED
- Yellow LED
- 220 ohms resistor

## Features

- Touch cycles through modes
- LCD displays active mode
- Simple fade loop
- All ON / All OFF
- Vacuum Tube display message
